# Api Starter Admin

> 这是一个极简的admin 管理后台。它只包含了 Element UI & axios & iconfont & 权限控制 & lint，这些搭建后台必要的东西。

## Build Setup

```bash
# 进入项目目录
cd api-starter-admin

# 安装依赖
npm install

# 启动服务
npm run dev
```

浏览器访问 [http://localhost:9528](http://localhost:9528)

## 发布

```bash
# 构建发布环境
npm run build

```
